# easyExcel

Excel 搜索与高亮工具

## 功能

- 搜索：支持多条件组合查询（AND/OR 逻辑）
- 条件类型：包含、等于、开头、结尾、大于、小于、正则、空值
- 高亮：单元格或整行高亮
- 导出：导出匹配结果到新文件

## 支持平台

- Windows (x86_64)
- macOS (x86_64)

## 使用

1. 选择 Excel 文件
2. 添加查询条件（列 + 类型 + 值）
3. 点击"开始搜索"
4. 设置高亮颜色
5. 点击"高亮并保存"或"导出结果"

## 构建

```bash
go build -o easyExcel ./cmd/main.go
```
